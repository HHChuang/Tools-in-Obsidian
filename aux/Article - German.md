

> **Workflow** - Reading, learning and writing.
> 1. Read the article, and highlight vocabulary or record the grammar point
> 2. Organise the vocabulary into a table in the section **Vocabulary list**
> 3. Use hotkey `cmd`+`shift`+`A` to generate an Anki card in Obsidian[^2] 
> 4. Sync the Anki note to the Anki app, which embeds the Obsidian note link in the note. 
> 5. Delete the Anki note in this Obaisian note[^1] 
> 6. **Writing practice** - rewrite the main topic or summarise it in 3-5 sentences
> 7. Feedback from AI - [LexiBot](https://www.lexibot.me/services/multilingual-grammar-checker/)

[^1]: since the plug-in only supports one-way sync but not two-way sync, it destroys the update I have when I use the anki app
[^2]: The hotkey from the Shell command plug-in executes the shell script `/Users/Grace/obsidian/meta/javascript/toAnkiCard.sh` .
